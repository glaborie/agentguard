# setglobalsnode properties #

![Set Globals node icon](https://dataplatform.cloud.ibm.com/docs/content/wsd/nodes/scripting_guide/clementine/images/setglobalsnodeicon.png)The Set Globals node scans the data and computes summary values that can be used in CLEM expressions\. For example, you can use this node to compute statistics for a field called `age` and then use the overall mean of `age` in CLEM expressions by inserting the function `@GLOBAL_MEAN(age)`\.
