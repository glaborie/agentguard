# timeplotnode properties #

![Time Plot node icon](https://dataplatform.cloud.ibm.com/docs/content/wsd/nodes/scripting_guide/clementine/images/timeplotnodeicon.png)The Time Plot node displays one or more sets of time series data\. Typically, you would first use a Time Intervals node to create a *TimeLabel* field, which would be used to label the *x* axis\.
