# derive\_stbnode properties #

![Space\-Time\-Boxes node icon](https://dataplatform.cloud.ibm.com/docs/content/wsd/nodes/scripting_guide/clementine/images/stbnodeicon.png) The Space\-Time\-Boxes node derives Space\-Time\-Boxes from latitude, longitude, and timestamp fields\. You can also identify frequent Space\-Time\-Boxes as hangouts\.
