# evaluationnode properties #

![Evaluation node icon](https://dataplatform.cloud.ibm.com/docs/content/wsd/nodes/scripting_guide/clementine/images/evaluationnodeicon.png)The Evaluation node helps to evaluate and compare predictive models\. The evaluation chart shows how well models predict particular outcomes\. It sorts records based on the predicted value and confidence of the prediction\. It splits the records into groups of equal size (quantiles) and then plots the value of the business criterion for each quantile from highest to lowest\. Multiple models are shown as separate lines in the plot\.
