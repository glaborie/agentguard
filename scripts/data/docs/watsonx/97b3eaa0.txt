# matrixnode properties #

![Matrix node icon](https://dataplatform.cloud.ibm.com/docs/content/wsd/nodes/scripting_guide/clementine/images/matrixnodeicon.png)The Matrix node creates a table that shows relationships between fields\. It's most commonly used to show the relationship between two symbolic fields, but it can also show relationships between flag fields or numeric fields\.
