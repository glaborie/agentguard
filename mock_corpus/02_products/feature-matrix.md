# Feature Matrix

## Core features by plan

| Feature | Starter | Business | Enterprise |
|---|---|---|---|
| Shared inbox | Yes (up to 3) | Yes (unlimited) | Yes (unlimited) |
| Workflow automation | Basic | Advanced | Advanced |
| CRM integrations (Salesforce, HubSpot) | No | Yes | Yes |
| SAML SSO | No | Yes | Yes |
| Audit logs | No | Yes | Yes |
| Role-based access control | Basic | Full | Full |
| Custom SLA | No | No | Yes (requires approval) |
| Dedicated onboarding | No | No | Yes |
| Premium support | No | No | Yes |
| Advanced security controls | No | No | Yes |
| IP allowlisting | No | No | Yes |
| Custom integrations | No | No | Yes (requires solution review) |

## Communication channels supported

NorthstarCRM handles inbound support and sales workflows via:
- Email (all plans)
- Web form submissions routed to shared inbox (all plans)
- Slack and Microsoft Teams notifications for routing and alerts (Business and Enterprise)

**Real-time live chat is not a native feature.** NorthstarCRM does not include a live chat widget or instant agent handoff capability. It is an asynchronous support and sales workflow platform. Teams requiring live chat should evaluate a dedicated live chat tool alongside NorthstarCRM.

## Automation depth

- **Basic (Starter)**: Single-step triggers (e.g., auto-assign by keyword, auto-reply on new ticket).
- **Advanced (Business and Enterprise)**: Multi-step workflows with conditional branching, SLA-based escalation triggers, and team-level routing rules.

## Integrations

- Salesforce: Bidirectional sync of contacts, accounts, and activities. Available on Business and Enterprise.
- HubSpot: Contact and deal sync. Available on Business and Enterprise.
- Slack: Notifications and ticket updates. Available on Business and Enterprise.
- Microsoft Teams: Notifications and ticket updates. Available on Business and Enterprise.
- Zendesk import tools: One-time data migration support available via onboarding team.
- Custom integrations: Available on Enterprise only, require solution design review.

## Add-ons (Enterprise only)

- Expanded data retention (beyond standard 2-year retention)
- Dedicated infrastructure (isolated environment)
- Additional admin seats beyond standard allocation
