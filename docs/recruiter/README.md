## Suggested verbal summary (30 seconds)

AgentGuard is an API-first reliability platform for RAG and agentic systems. It integrates model gateway routing, observability, and evaluation controls with explicit mitigation for runtime risks such as prompt injection, data leakage, and quality regressions. The implementation is supported by repeatable tests and commit-visible hardening over time.
