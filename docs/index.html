Full-stack observability

End-to-end traces, retrieval latency, tool usage, and model behavior — all integrated with OpenTelemetry, Langfuse, Grafana, and Jaeger.

  • Traces for every RAG & agent step
  • Langfuse dashboards + Arize AX tracing
  • Real-time latency & token usage metrics

Runtime guardrails & compliance

Prompt injection blocking (regex + LLM-judge), toxic content detection, PII masking, and agent tool-call guardrails. Stop data leaks before they reach customers.

  • Real-time prompt injection detection
  • Automatic PII redaction
  • Tool call authorization & blocking

Golden datasets & regression checks

Prevent regressions after prompt, model, retrieval, or tool changes. Run benchmarks, score faithfulness, and track release confidence.

  • Golden dataset evaluation
  • Automated scoring & regression alerts
  • CI-compatible test suites

Automated adversarial probing

Simulate 4 attack types: prompt injection, jailbreak, PII extraction, and system prompt leak. CI-friendly exit codes for pre-deployment hardening.

  • Red-team automatically before release
  • Hardened system prompt generation
  • Detect weaknesses in tool orchestration

Launch AgentGuard in minutes
# 1. Clone the repository
git clone https://github.com/glaborie/agentguard.git
cd agentguard

# 2. Configure environment
cp .env.example .env

# 3. Spin up the full stack (Langfuse, Qdrant, Ollama, Redis, Grafana...)
docker compose up -d

# 4. Pull local embedding model
docker compose exec ollama ollama pull nomic-embed-text

# 5. Ingest knowledge & query
pip install -r requirements.txt
python -m app.main ingest
python -m app.main query "Does the Starter plan include SAML SSO?"
→ Open WebUI: http://localhost:3100
→ Langfuse UI: http://localhost:3200
